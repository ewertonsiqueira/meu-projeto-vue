# 📖 Bem-vindo ao Storybook da RotaExata!

Este Storybook foi criado para documentar e testar visualmente os componentes da RotaExata. Aqui, você encontrará exemplos interativos, variações e estados dos nossos componentes para facilitar o desenvolvimento e a manutenção do projeto.

## 🚀 Como Usar
1. **Navegue** pelo menu lateral para explorar os componentes.
2. **Interaja** com as propriedades para testar diferentes variações.
3. **Integre** os componentes conforme necessário no seu projeto.

Aproveite e boas criações! 🎨✨
